# Stelpro Drivers

Hubitat Elevation drivers for Stelpro (and the Hilo / Allia OEM variants) Zigbee thermostats.

## Drivers

- `StelproKi.groovy` — **Stelpro Ki ZigBee Thermostat**: Zigbee thermostat for the Stelpro Ki line
- `StelproAllia.groovy` — **Stelpro Allia Zigbee Thermostat**: Zigbee thermostat for Allia / Stello Hilo HT402

## Reference notes

- `Stelpro_orleans_specs.groovy` — Zigbee node-descriptor reference for the Stelpro Orleans hardware (commented reference data; not a driver)
